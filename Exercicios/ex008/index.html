Formatação Negrio e Itálico

Negrito

quando quisermos dar destaque a uma palavra usamos a tag <B> (Não semântico).

E aquando queremos dar o destaque de forma que tenha sentido usamos a tag <strong> (Semântico).


Itálico

Quando queremos dar enfase a um texto usamos a tag <i> (não semântico).

Quando queremos dar enfase de formar certa usamos a tag <em> (Semântico).

Texto marcados

Podemos marcar textos usando a tag <mark>.

Texto grande e pequeno

Para deixar o texto grande usamos a tag <big>(mas ela já está obsoleta, faça isso usando css.), e para deixar o texto pequeno usamos a tag <small>.

Texto excluído

Para deixar o texto deletado usamos a tag <del>

Texto inserido

Para subilihar um texto usamos a tag <ins>

Texto sobrescrito

Para sobrescrever um texto usamos a tag <sup> H20c

Texto subscrito

Para sobscrever um texto usamos a tag <sub> H2O
